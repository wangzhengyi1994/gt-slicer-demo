# GT Slicer Demo - ViewCube 视觉增强

## 任务
增强右上角 SVG ViewCube 的视觉效果,让它更精致专业

## 当前状态
- 纯 SVG 实现, 6 个面(前后左右上下), 有 hover 变色, 点击切换视角
- 视觉: 纯色填充 + 文字标签, 比较朴素

## 需要增强的视觉效果

### 1. 面的渲染
- 每个面添加微妙的渐变(不是纯色,从一个角到另一个角有轻微明暗变化)
- 面之间的棱边更清晰,增加 1px 的深色描边
- 当前面(正对摄像机的面)比侧面更亮

### 2. 边缘和棱角
- 12 条棱边: 增加高亮效果,像是被光照的金属边缘
- 8 个角点: 添加小圆点装饰(2px 圆, 用于角点定位标识)
- 棱边区域也可以悬停,悬停时高亮并显示对应的棱边方向名(如"前上", "右下")

### 3. 轴线指示器
- ViewCube 下方或旁边添加三色轴线指示器 (mini XYZ axes)
- 红色短线 = X, 绿色短线 = Y, 蓝色短线 = Z
- 轴线随 ViewCube 旋转同步

### 4. 悬停效果增强
- 面悬停: 添加辉光边框(glow effect, 用 SVG filter)
- 面悬停时文字加粗 + 放大一点
- 悬停过渡要有动画(opacity 和 fill 的 CSS transition)

### 5. 8个角标签
- 除了6个面中心的"前后左右上下", 在接近角点的位置添加组合标签
- 如: 前+上+右 的角附近显示"前上右" (当鼠标接近角时)
- 边和角的标签只在 hover 时显示

### 6. ViewCube 尺寸
- 从 120px 增大到 140px,给更多细节空间

### 7. 背景
- ViewCube 区域添加圆形半透明背景(毛玻璃效果)
- 深色主题: 暗色半透明
- 浅色主题: 白色半透明

## 技术要求
- 仍然是纯 SVG 实现(在 app.js 的 _viewCube 模块中修改)
- 可以用 SVG filter 实现 glow/shadow
- 保持所有现有功能(点击切换视角, 与相机同步, hover 变色)
- 不影响其他功能
